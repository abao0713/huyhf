# 缠论量化交易策略实现计划

## 一、项目概述

基于缠论（Chan Theory）的量化交易策略，实现30分钟级别的主交易周期配合日线定方向的联动交易系统。

### 核心目标
- 30分钟K线识别笔、线段、分型
- 结合MACD判断背驰
- 日线辅助判断方向
- 自动生成交易信号并执行

```python
        return {"action": "BUY", "stop_loss": last_pen['low'], "reason": "30分钟底背驰"}

    if self.is_top_divergence(last_pen):
        return {"action": "SELL", "stop_loss": last_pen['high'], "reason": "30分钟顶背驰"}
```

```python
class ChanStrategyExecutor:
    def __init__(self, client: BinanceRestClient):
        self.client = client
        self.strategy = ChanStrategy()

    async def run(self, symbol: str):
        """
        主循环：
        1. 获取数据
        2. 计算信号
        3. 执行交易
        """
        while True:
            # 获取30分钟和日线数据
            df_30m = await get_klines(symbol, '30m')
            df_daily = await get_daily_klines(symbol)

            # 计算缠论指标
            self.strategy.update(df_30m, df_daily)

            # 生成信号
            signal = self.strategy.generate_signal()

            # 执行交易
            if signal['action'] != 'HOLD':
                await self.execute_trade(signal)

            await asyncio.sleep(60)  # 每分钟检查一次
```

## 七、注意事项

1. **防止未来函数**：分型确认需要等后续K线走出来才能确认
2. **信号闪烁**：避免在分型未确认时频繁切换信号
3. **参数适配**：30分钟级别建议HG1=8，可根据交易品种调整
4. **风控优先**：始终设置止损，控制单笔亏损

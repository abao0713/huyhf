# 多周期共振底分型预判策略 Spec

## Why
当前V2策略仅基于30分钟单一周期分型驱动交易，缺乏多周期共振验证和支撑位过滤，容易在市场噪音中产生假信号。需要一套结合30分钟+15分钟双周期共振、关键支撑位验证、多指标确认的底分型预判策略，提高做多信号的准确性和风控能力。

## What Changes
- 新增 `MultiTFFractalStrategy` 策略类，继承 `BaseStrategy`
- 新增 `MultiTFFractalStrategyExecutor` 执行器，处理双周期数据获取和交易执行
- 新增 `run_ethusdc_mtf_live.py` 运行脚本
- 支持30分钟+15分钟双周期K线数据获取
- 支持关键支撑位列表注入（水平线/趋势线/斐波那契）
- 实现试探性入场（40%）+ 确认加仓（60%）的两阶段仓位管理
- 实现多层风控：单笔2%止损、连续3次止损暂停、日亏损5%停止

## Impact
- Affected specs: 新增独立策略，不影响现有V1/V2策略
- Affected code: 
  - `trading_system/strategies/mtf_fractal_strategy.py` (新增)
  - `run_ethusdc_mtf_live.py` (新增)
  - `trading_system/strategies/__init__.py` (修改)

## ADDED Requirements

### Requirement: 30分钟级别支撑位预判
系统 SHALL 在30分钟K线图上检测价格是否进入关键支撑区域（当前价格 <= 支撑位 + 阈值），并识别潜在底分型结构（K1、K2两根K线满足K2.low < K1.low且K2.close > K2.open * 0.5）。

#### Scenario: 价格进入支撑区域且出现底分型雏形
- **GIVEN** 用户配置了关键支撑位列表 [2250, 2200, 2150]
- **WHEN** 30分钟当前价格 <= 2250 + 阈值(如10点) 且 存在K1,K2满足底分型条件
- **THEN** 系统切换到15分钟图表进一步确认

#### Scenario: 价格远离支撑区域
- **GIVEN** 30分钟当前价格 = 2350，最近支撑位 = 2250
- **WHEN** 价格远高于支撑位
- **THEN** 系统不触发后续15分钟确认流程

### Requirement: 15分钟级别多信号确认
系统 SHALL 在15分钟图表上检测以下4类信号，至少满足2个才触发试探性入场：
1. **底背离**: 15分钟价格创新低但MACD未创新低
2. **看涨K线形态**: 出现早晨之星、看涨吞没或锤子线
3. **突破下降趋势线**: 收盘价突破最近N根K线的下降趋势线
4. **指标金叉**: MACD金叉或KDJ低位金叉(K<20)

#### Scenario: 满足2个信号触发试探性入场
- **GIVEN** 15分钟图表检测到底背离 + MACD金叉
- **WHEN** 满足信号数 >= 2
- **THEN** 系统生成试探性入场信号，仓位=总仓位40%，止损=min(15分钟信号K线最低点, K2最低点) - 偏移

#### Scenario: 满足1个信号不触发
- **GIVEN** 15分钟图表仅检测到底背离
- **WHEN** 满足信号数 < 2
- **THEN** 系统不生成入场信号，等待更多确认

### Requirement: 两阶段仓位管理
系统 SHALL 支持试探性入场（40%仓位）和确认加仓（60%仓位）两阶段操作。

#### Scenario: 试探性入场后30分钟K3确认
- **GIVEN** 已执行试探性入场（40%仓位）
- **WHEN** 30分钟K3收盘价 > K2最高价
- **THEN** 系统生成加仓信号，加仓仓位=总仓位60%，止损更新为底分型最低点 - 偏移

#### Scenario: 试探性入场后K3未确认
- **GIVEN** 已执行试探性入场
- **WHEN** 30分钟K3收盘价 <= K2最高价
- **THEN** 系统保持现有仓位，不执行加仓，继续监控

### Requirement: 多层风控规则
系统 SHALL 执行以下风控规则：
1. 单笔最大亏损 <= 总资金2%
2. 连续3次止损后暂停交易
3. 每日最大亏损达总资金5%时停止当日交易

#### Scenario: 单笔亏损超限
- **GIVEN** 总资金 = 10000 USDT
- **WHEN** 计算出的止损价导致单笔亏损 > 200 USDT
- **THEN** 系统调整仓位大小使亏损 <= 200 USDT

#### Scenario: 连续3次止损
- **GIVEN** 近期已发生3次连续止损
- **WHEN** 新信号生成
- **THEN** 系统忽略信号，暂停交易直到手动重置

#### Scenario: 日亏损达5%
- **GIVEN** 当日累计亏损已达总资金5%
- **WHEN** 新信号生成
- **THEN** 系统停止当日交易，次日自动恢复
